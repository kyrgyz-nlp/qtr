# 🐭 Qytyr qytyr

Kiril tamğasynan Qyrğyz-latyn tamğasyna transliterlöö.

## 🤌 Qoldonuu

### 🛠️ Quruu

```sh
go build .
```

### 🔮 Tekstterdi transliterlöö

```sh
cat FILE.txt | ./qtr
echo "Салам дүйнө" | ./qtr
```
